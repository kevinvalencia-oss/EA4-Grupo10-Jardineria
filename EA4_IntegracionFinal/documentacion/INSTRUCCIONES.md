# EA4 — Instrucciones para el Entregable

## Archivo a entregar
Nombre: `EA4_Grupo10_Integracionfinal.pdf`

## Contenido del PDF
1. **Portada** con los datos del estudiante y del curso
2. **Enlace al repositorio de GitHub** (pegar aquí una vez publicado)

## Pasos para publicar el repositorio en GitHub
1. Crear repositorio en https://github.com/new con nombre `EA4-Grupo10-Jardineria`
2. Desde la carpeta del proyecto:
```bash
git remote add origin https://github.com/TU_USUARIO/EA4-Grupo10-Jardineria.git
git push -u origin main
```
3. Copiar la URL del repositorio y pegarla en el PDF de portada.
